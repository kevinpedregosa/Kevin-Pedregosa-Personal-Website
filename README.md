# Kevin Pedregosa Personal Website

This repository contains my personal portfolio website, including:
- `About` (homepage)
- `Projects`
- `Experience`

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla + jQuery)

## Project Structure
- `index.html` - About page
- `projects.html` - Projects page
- `experience.html` - Experience page
- `menu.html` - Shared navigation menu partial
- `css/frame.css` - Base framework styles
- `css/custom.css` - Custom theme and page styling
- `js/menu.js` - Menu loading/behavior
- `img/` - Images and logos used across the site
